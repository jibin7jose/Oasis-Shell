# 🌬️ OASIS SHELL: Neural Executive Blueprint

Oasis Shell is a sentient, AI-native operating layer designed to transform your desktop from a static file explorer into a proactive, executive workspace. It bridges the gap between local system performance (Rust) and high-level neural reasoning (Local LLMs).

---

## 🏗️ 1. Technical Architecture (Triple-Aura)

The system is built on a distributed bridge architecture:

### **A. Neural Executive Layer (Rust/Tauri)**
- **Role:** The "Brainstem." Handles physical file access, process monitoring, and secure PowerShell execution.
- **Key Modules:**
  - `start_proactive_sentience`: A background thread monitoring hardware health (RAM/CPU).
  - `execute_neural_command`: A secure wrapper that translates AI intent into local commands.
  - `Neural Heartbeat`: A real-time diagnostic engine that pings local models (Gemma3) and ensures the link is stable.

### **B. Fluid Interface (React/Vite)**
- **Role:** The "Consciousness." A glassmorphic, 3D-accelerated UI that visualizes your system state.
- **Components:**
  - **Neural Vault:** A semantic file graph where apps are "Crated" into context-aware workspaces.
  - **Telemetric Pulse:** Live data visualization from local services (Scout-Racer).
  - **Command Horizon:** The chat-driven executor that detects user intent and suggests actions.

### **C. Local Neural Engine (Ollama)**
- **Brain:** `gemma3:4b` - Used for command generation and reasoning.
- **Eyes:** `nomic-embed-text` - Used for RAG (Retrieval Augmented Generation) and semantic filing.
- **Resilience:** All intelligence is **100% offline.** Your data never leaves your D-drive.

---

## 🛡️ 2. Core Functional Pillars

### **I. Proactive Sentience**
Oasis doesn't wait. If your system RAM exceeds 85%, the background monitor emits a `proactive-pulse`. The UI will slide up a suggestion: *"I notice high memory load. Should I crate your inactive workspaces?"*

### **II. Neural Crating**
Instead of folders, Oasis uses **Crates**. 
1. The AI scans your open windows (`get_running_windows`).
2. Gemini/Gemma suggests a context-aware name (e.g., "Neural Forge Research" instead of "Untitled Project").
3. One-click restoration of your entire workspace.

### **III. The Execution Bridge**
When you type a command (e.g., "how is my git status?"):
- Oasis processes the intent locally.
- It generates a secure [CMD] suggest hook.
- You click **RUN**, and the shell executes the PowerShell logic natively.
